# AGENT — Esports Coverage Network

A serious source for esports coverage. ESPN-style live scores, player profiles, team profiles, tournament archives, and full historical timeline. Built Fortnite-first, designed to expand to Counter-Strike, Valorant, League of Legends and more.

## Pages

- **Home** (`index.html`) — Live score strip, hero story, news grid, Power Rankings, schedule
- **Players** — Directory plus 13 individual wiki-style profiles (Peterbot, Pollo, Bugha, Khanada, Mongraal, MrSavage, EpikWhale, Ninja, Tfue, Clix, Aqua, Nyhrox, MariusCOW)
- **Teams** — Directory plus 5 team wiki pages (Falcons, Dignitas, FaZe, NRG, 100 Thieves)
- **Tournaments** — Index plus archive pages (2019 World Cup, FNCS 2024 Global Championship)
- **Live Coverage** — Multi-region FNCS standings and game tracker
- **History** — Full Fortnite competitive timeline 2018 to 2026
- **Stats** — Career earnings, K/D, win rate, FNCS placement leaderboards
- **Compare** — Side-by-side player comparison
- **Coach** — Coach profile pages
- **News** — Long-form article layout

## Search

Press `Cmd/Ctrl+K` or `/` anywhere on the site to open the global search overlay. 28 indexed entries across players, teams, tournaments, and pages.

## Photo and data sources

Player photos via Liquipedia (CC-BY-SA 3.0) with attribution in every page footer. Atmospheric photography via Unsplash. Statistical data sourced from Liquipedia Fortnite Wiki, Esports Earnings, Wikipedia.

## Stack

Static HTML, single shared `styles.css`, single shared `app.js`. No build step, no framework. Deploy as static files anywhere.
